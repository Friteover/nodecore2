# NODECORE

A modern knowledge workspace inspired by Obsidian, VSCode, Linear and Notion.

────────────────────────────────────────────────────

🚀 OVERVIEW

NODECORE is a lightweight, offline-first note-taking and knowledge management system built entirely with vanilla web technologies.

It helps you think in connections, not files.

────────────────────────────────────────────────────

⚡ FEATURES

• Markdown editor with live preview  
• Wikilinks system [[note]]  
• Backlinks auto-generation  
• Interactive knowledge graph  
• Command palette (CTRL + K)  
• Fast search engine  
• LocalStorage persistence  
• Dark premium UI  
• Responsive layout (desktop/mobile)

────────────────────────────────────────────────────

🧠 KNOWLEDGE GRAPH

Turn your notes into a living network.

• Nodes = notes  
• Edges = links  
• Dynamic visualization  
• Zoom / pan / focus mode  

────────────────────────────────────────────────────

⌨️ SHORTCUTS

CTRL + K → Command palette  
CTRL + N → New note  
CTRL + G → Toggle graph view  

────────────────────────────────────────────────────

🎨 UI / UX

Designed with a modern aesthetic:

• Dark-first interface  
• Glassmorphism effects  
• Smooth animations  
• Minimal & distraction-free  
• VSCode-style workflow  

────────────────────────────────────────────────────

🛠 STACK

• HTML5  
• CSS3  
• Vanilla JavaScript  
• Cytoscape.js  
• Marked.js  
• Highlight.js  

────────────────────────────────────────────────────

💾 STORAGE

• LocalStorage (offline-first)  
• Auto-save system  
• No backend required  

────────────────────────────────────────────────────

🔥 GOAL

Build a fast, elegant and extensible alternative to Obsidian
directly in the browser — no framework, no backend.
